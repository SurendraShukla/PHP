#### Start in shell WebSocket server 
`php -q websockets.php`

#### Start in shell WebSocket client 
`php -S 0.0.0.0:8000 websockets.html`

#### Open in browser URL 
`http://127.0.0.1:8000/`
 
You will see current timestamp which will continuously updating!
